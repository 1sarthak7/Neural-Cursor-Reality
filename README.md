# Neural Cursor Reality

A cinematic WebGL experiment where your cursor transforms into a glowing neural energy orb.

It reacts to:

* Mouse movement
* Proximity to UI elements
* Real-time microphone frequencies

This is not a UI.
This is a distortion engine.

## Core Features

| System                   | Description                               |
| ------------------------ | ----------------------------------------- |
| Neural Energy Orb        | Custom GLSL shader-driven glow core       |
| Liquid Light Pulse       | Time-based animated fragment distortion   |
| Magnetic UI              | DOM elements bend toward cursor proximity |
| Unreal Bloom             | Postprocessing cinematic glow             |
| ACES Filmic Tone Mapping | Premium color grading                     |
| Sound Reactive Intensity | Web Audio API frequency analysis          |

## Architecture Overview

```text
Mouse Input  →  Three.js Scene  →  Shader Material  →  Bloom Composer
                ↓
         Web Audio API → Frequency Data → Glow Intensity
                ↓
         Magnetic DOM Distortion
```
